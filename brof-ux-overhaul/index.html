BRoF UX Overhaul

Business Review of the Future (BRoF) is a web app that doubles as a business analytics tool and a presentation (deck creation) tool for senior-level managers at Colgate-Palmolive.

This requirement poses a few challenges -- we have to fit a substantial amount of elements above the fold to provide an at-a-glance overview of the business, and ensure each page is still presentable on larger displays.

Role
Web Design, UX Design

  • Background & approach

    When I was first brought onto the team, BRoF was mid-way through a major UI overhaul. This was mainly due to the added requirement that is has to double as a presentation tool.

    After being briefed on the project, it was clear that I would not have much say over the content of each page. Not only that, as the application was still under redevelopment, many elements such as deck-building and filtering* were not functional yet.

    Hence, I decided to focus on the things I could manipulate: the landing page, the general layout of the dashboard, and the navigation menu.

    * I was requested to design the filtering flow a couple of months later and my work on that can be found here.

  • Landing Page

    We were certain thats users launch this application to either view/edit an item or create an item. An item here would be either an analytics dashboard (a story) or a presentation deck.

    The new landing page differentiates between stories and presentations, and for both there are items created by the user, and items created and published by other employees. After the distinction was made, it made sense to employ a two-column layout, and order the subgroups by priority.

    Visually, significant emphasis was also added to the 'Create a ...' buttons and a thumbnail was included in each tile in place of a chart icon.

  • General Layout

    Visual hierarchy

    We enlarged the text for the higher-level items "Finance", "Marketing", and "Macroeconomics", and gave them a clearer bounding box to better show hierachy and grouping. Then, the "as of month" was moved to the immediate right of the category text for a natural flow.

    Proximity of filtering elements

    There can be up to three filter groups such as 'Geography' in the top-left. Filtering with these groups and using the 'Period' dropdown menu on the top-right do very similar things from the user's perspective, yet they were unintuitively placed at opposite ends of the screen.

    After a brief survey, we gathered that the globe icon along with the 'Filter' label conveys a geography filter clearly enough, so we removed its label to reduce clutter. All that was left to be done here was to put the filtering icons next to the dropdown period filter.

    Other minor changes

    We also increased font size for better legibility, removed unncessassary text decorations, and page elements were condensed to display more above the fold.

    Some inconsistencies, such as the positioning of year-over-year deltas in relation to the key performance indicator's current value, and the color representation of deltas (good vs. bad change), were flushed out.

  • Navigation Menu

    Icon button

    In the first version of BRoF, the developers received feedback that their users did not understand the hamburger icon to mean 'menu', so they added a label under it. However, the label was much too tiny, so we enlarged it and pulled it to the right.

    Menu items

    The first thing we did was implement a darker color scheme for the menu so that it stands out from rest of the site. To indicate the current page in the hierarchy, we only highlighted the item with blue, and did away with the indenting.

    We also decided to collapse all other branches aside for the current navigation tree to reduce clutter. To help with this, triangular expand and collapse bullets were used.

  • Conclusion

    This project was very interesting in the sense that we had to wrestle with not having control over the content, the type of charts used, or the fact that we had to come up with a design that can effectively populate two different screen real estate and deliver impactful data analysis.
